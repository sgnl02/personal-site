My portfolio consists (mostly) out of experimental code, small snippets or code I have been experimenting with.

JavaScript

You most likely will be needing the web console (for Chrome or FireFox) or Dragonfly (for Opera)

  • Augmenting types: useful for making methods available to all functions.
  • Closures: having access to outer variables is useful to create private scopes.
  • Do Not Track: reading DNT-headers with JavaScript (DNT is unreliable).
  • Function invocation pattern: using this in a more reliable way by storing it in that.
  • Global abatement: creating properties for one local variable to reduce global variables.
  • Lambda: passing a function as an argument.
  • Module: making a module out of a method.
  • Not operator boolean: !! produces booleans.
  • Parseint: it's recommended to use the radix parameter to prevent problems with leading zeros.
  • Shorthand if-statements: the or-operator can be used as an if-statement.
  • Click-to-Retweet: an improvement over static shared lists of Tweets which now can be clicked.
  • "Circos Sin Animales" (Experimental): map with places showing where it is not allowed to have circuses with animals in México.
  • Vegamap Monterrey (Experimental): map that shows vegetarian/vegan places in Monterrey.
